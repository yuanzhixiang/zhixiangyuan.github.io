该方法来自 Netty 源码

```java
    /**
     * @param val 需要被判断是否是 2 的幂次的数
     * @return true 是 2 的幂次 / false 不是 2 的幂次
     */
    public static boolean isPowerOfTwo(int val) {
        return (val & -val) == val;
    }
```
