比如说想要对 4 取余，那么下面这段代码便可以做到

```java
    public static int mod4(int num) {
        return num & 0b0000_0011;
    }
```

对一个 2 的幂次的数做取余操作可以使用下面的这套公式

```java
    public static int mod4(int num) {
        return num & (2^幂次 - 1);
    }
```

稍微改造一下便可实现别的功能，比如说是否整出 4

```java
    public static boolean mod4(int num) {
        return (num & 0b0000_0011) == 4;
    }
```
