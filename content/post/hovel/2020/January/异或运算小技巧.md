# 1 自己与自己运算得到 0

```java
    // 当 x = y 时， x ^ y = 0
    public static int method(int x, int y) {
        return x ^ y;
    }
```

# 2 在一定条件下使用异或实现减法效果

```java

    // 间的时候异或都是减法的效果
    // 1024 = 0b00000100_00000000
    // 2047 = 0b00000111_11111111  ^
    // ------------------------------
    // res  = 0b00000011_11111111
    // 最高位相同为 0 抵消掉了
    // 
    // 那么条件就是其中一个数 x 为 2 的幂次
    // 另一个数的范围在 y = [x, 2 * x - 1] 之间
    // 这样这两个数的异或结果便是减法的效果
```
