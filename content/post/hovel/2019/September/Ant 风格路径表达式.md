Ant 风格路径表达式就是一个精简版的正则表达式

| 通配符 | 说明                      |
| ------ | ------------------------- |
| ?      | 匹配任何单字符            |
| *      | 匹配 0 或者任意数量的字符 |
| **     | 匹配 0 或者更多的目录     |
