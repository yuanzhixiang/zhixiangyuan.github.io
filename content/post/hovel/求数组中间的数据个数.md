# 1 求范围内的数字个数

求 0 和 4 中间的数字个数

## 1.1 既不包含 0，也不包含 4，即 (0,4)

`4 - 0 - 1 = 3`

## 1.2 包含 0 或 4，即 (0,4] 或是 [0,4)

`4 - 0 = 4`

## 1.3 既包含 0 又包含 4，即 [0,4]

`4 - 0 + 1 = 5`
