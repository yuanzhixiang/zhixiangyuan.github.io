1 前言

本文介绍 int 数值入栈指令 iconst、bipush、sipubh、Idc。

根据 int 取值不同分为以下几种可能性：

  1. iconst: [-1, 5]
  2. bipush: [-128, -2]∪[6, 127]
  3. sipush: [-32768, -129]∪[128, 32767]
  4. ldc: [-2147483648, -32769]∪[32768, 2147483647]

2 iconst

当 int 取值 [-1, 5] 时，JVM 采用 iconst 指令将常量压入栈中。

  6. iconst_4: 4
  7. iconst_5: 5
