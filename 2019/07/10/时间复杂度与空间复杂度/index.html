1 时间复杂度

分析程序时间复杂度的时候只需要关注循环中的代码即可

1.1 常见的时间复杂度

常见的时间复杂度有 O(1), O(n), O(logn), O(nlogn), O(\(n^2\)).. O(\(n^k\)), O(\(2^n\)), O(n!)

其中 O(\(2^n\)) 和 O(n!) 在数据规模增大时呈现指数级增长，属于很糟糕的表现

    int cal(int m, int n) { 
        int sum_1 = 0; 
        for (int i = 1; i < m; ++i) { 
            sum_1 = sum_1 + i; 
        }
        int sum_2 = 0;  
        for (int j = 1; j < n; ++j) { 
            sum_2 = sum_2 + j; 
        }
        return sum_1 + sum_2;
    }

时间复杂度：O(m + n)

2 空间复杂度

    void print (int n) {
        int[] a = new int[n];
        for(int i = 0; i < n; i++) {
            a[i] = i * i;
        }
    }

空间复杂度：O(n)

上述代码只有 int[] a = new int[n] 申请了一个大小为 n 的 int 类型数组，剩下的没有占用更多的空间，所以整段代码的空间复杂度就是 O(n)。
